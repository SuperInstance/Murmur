# Murmur

Self-populating TensorDB wiki and bulletin board for SuperInstance.

## Overview

Murmur is a knowledge management system that organizes itself. Built on the TensorDB knowledge graph, it provides:

- **Self-Populating Wiki** - Automatically organizes information
- **Bulletin Board** - Community-driven updates
- **Knowledge Graph** - TensorDB-powered connections
- **Real-time Updates** - Live event streaming

## Features

- **Auto-Organization** - Content structures itself
- **Semantic Connections** - AI-powered linking
- **Community Features** - Boards and threads
- **TensorDB Integration** - Knowledge tensor operations

## Quick Start

```bash
git clone https://github.com/SuperInstance/Murmur.git
cd Murmur
pnpm install
pnpm dev
```

Visit http://localhost:3004

## Development

```bash
# Install dependencies
pnpm install

# Run dev server
pnpm dev

# Build for production
pnpm build

# Type check
pnpm type-check
```
